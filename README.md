
This is a [React Native App](https://reactnative.dev/) 


# App Name: Todo App



## To run

First, fork or clone this repo then run:

```bash
npm install
```
To install all dependencies

After that, run:
```bash
expo start
```
and connect to your android studio or expo client on a physical device
